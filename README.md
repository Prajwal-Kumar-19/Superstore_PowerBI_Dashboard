  # 🚀 Superstore Analysis - Power BI Dashboard 🛒📊

This project showcases an interactive Power BI dashboard built on a Superstore dataset, aimed at analyzing sales performance, profitability, and customer trends. The goal was to convert raw Excel data into meaningful insights through effective data transformation and visualization.

# 🔧 Data Preparation & Transformation

Imported raw data from Excel into Power BI

Cleaned and transformed data using Power Query 🧹

Handled missing values, corrected data types, and structured the dataset for accurate analysis

Created calculated columns and measures for deeper insights

# 📊 Dashboard Features
📈 Sales & Profit Analysis: Visualized revenue, profit, and key KPIs across regions and categories

🌍 Regional Insights: Compared performance across different locations

📦 Category & Sub-category Breakdown: Identified top and underperforming product segments

🧑‍💼 Customer Insights: Analyzed customer purchasing behavior and contribution

🖱️ Interactive Tooltips: Provided additional context on hover

🔍 Drill-through Pages: Enabled detailed exploration of specific segments

🎛️ Buttons & Navigation: Smooth and user-friendly navigation across dashboard views

🎨 Clean & Modern UI: Designed with a focus on clarity, readability, and visual appeal

# 🔍 Key Insights
Identified high-revenue regions and top-performing product categories 📊

Discovered areas with high sales but low profitability ⚠️

Highlighted customer segments contributing the most to revenue 💡

Tracked trends and patterns to support better business decisions

This dashboard transforms static data into an interactive analytical tool, allowing users to explore business performance, uncover hidden trends, and make data-driven decisions effectively. 🚀
